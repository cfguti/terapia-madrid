# terapia-madrid
Laura Baños del Río es psicóloga y psicoanalista en Madrid. Ofrece psicoterapia para adultos, adolescentes y parejas en su consulta de Madrid y también en formato online. Acompaña procesos relacionados con ansiedad, conflictos personales, maternidad, relaciones y malestar emocional desde un enfoque cercano y profesional.
